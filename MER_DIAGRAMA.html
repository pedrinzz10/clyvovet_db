Como gerar o modelo no Oracle Data Modeler

  1. Abra o Oracle Data Modeler
  2. Vá em File > Import > DDL File
  3. Selecione o arquivo 07_ORACLE_MODELER_IMPORT.sql
  4. Clique em Generate — o Modelo Relacional será criado automaticamente
  5. Para o Modelo Lógico: no Navigator, clique com o botão direito em Relational Models > Engineer to Logical Model
  6. Para o Diagrama Físico (DER): abra o modelo relacional e acesse View > Logical Diagram
